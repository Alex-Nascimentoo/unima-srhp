# Sistema de Recomendação Hierárquica de Produtos (SRHP)

Projeto utilizando API Flask

## Setup do projeto para desenvolvimento:

Esse projeto utiliza o gerencidor de dependências Poetry, você pode instalá-lo [aqui](https://python-poetry.org/docs/#installation)

### 1. Instalando dependências

  ```bash
  poetry install
  ```

### 2. Rodar aplicação Flask

```bash
  poetry run start
  ```

  Por padrão, o app vai estar disponível em ```http://127.0.0.1:5000```
